# 🌿 Calmaria SPA  

Este projeto é uma simulação de um site para o **Calmaria SPA**, desenvolvido durante o curso de **SASS e CSS: Estilizando um site** na Alura. Durante o curso, aprendemos a criar estilos elegantes e sofisticados utilizando SASS, com foco em organização de código e reutilização de componentes.

🔗 [Acesse o site aqui](https://davidtpires.github.io/calmaria-spa/)

## 🎨 Visão Geral  

- Design moderno e relaxante, alinhado com a identidade visual de um SPA.  
- Estilos criados utilizando SASS para maior eficiência e manutenção.  
- Desenvolvimento centrado em boas práticas de organização de código e responsividade.

## 🛠️ Tecnologias Utilizadas  

- **HTML5:** Estruturação semântica e acessível.  
- **CSS3 e SASS:** Modularização e criação de estilos reutilizáveis.  
- **GitHub Pages:** Publicação e hospedagem do projeto.
